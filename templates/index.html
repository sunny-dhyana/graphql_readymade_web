GraphQL ReadyMade Web

Welcome!

GraphQL ReadyMade Web is a simple demonstration application built using GraphQL. It provides a ready-to-use backend API along with a web interface to explore queries, mutations, and subscriptions.


Getting Started

This application exposes a GraphQL endpoint that you can interact with using the built-in UI or any GraphQL client such as Apollo, Insomnia, or curl.

Use the navigation menu to explore available features and try out different queries. The API supports standard GraphQL operations and follows common schema conventions.


Application Modes

The application supports multiple operating modes that can be configured globally. These modes control how the server processes and validates GraphQL requests.

You can switch modes from the top bar menu or by sending the HTTP header X-GRMW-MODE with your requests.

If no mode is specified, the application will use the default configuration.


Mode Overview

Standard

The Standard mode provides a straightforward GraphQL setup with minimal constraints, suitable for development and experimentation.

Advanced

The Advanced mode enables additional query validation and performance controls, such as depth limits and request analysis, making it more suitable for production-like usage.
